Deploy Maumau on Northflank with a one-click template and access it through the web Control UI.
This is the easiest "no terminal on the server" path: Northflank runs the Gateway for you.

## How to get started

1. Click [Deploy Maumau](https://northflank.com/stacks/deploy-maumau) to open the template.
2. Create an [account on Northflank](https://app.northflank.com/signup) if you don't already have one.
3. Click **Deploy Maumau now**.
4. Set the required environment variable: `MAUMAU_GATEWAY_TOKEN` (use a strong random value).
5. Click **Deploy stack** to build and run the Maumau template.
6. Wait for the deployment to complete, then click **View resources**.
7. Open the Maumau service.
8. Open the public Maumau URL at `/maumau` and connect using your `MAUMAU_GATEWAY_TOKEN`.

## What you get

- Hosted Maumau Gateway + Control UI
- Persistent storage via Northflank Volume (`/data`) so config/credentials/workspace survive redeploys

## Connect a channel

Use the Control UI at `/maumau` or run `maumau onboard` via SSH for channel setup instructions:
