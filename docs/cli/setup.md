# `maumau setup`

Initialize `~/.maumau/maumau.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- CLI onboarding: [Onboarding (CLI)](/start/wizard)

## Examples

```bash
maumau setup
maumau setup --workspace ~/.maumau/workspace
```

To run onboarding via setup:

```bash
maumau setup --wizard
```
