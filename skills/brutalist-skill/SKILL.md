# Brutalist Skill

Adapted for Maumau from the public `leonxlnx/taste-skill` repo.

Use this for deliberate harshness, not accidental ugliness.

- favor rigid structure, hard hierarchy, and utilitarian restraint
- keep surfaces crisp and purposeful instead of decorative
- use only when the brief genuinely wants mechanical severity
